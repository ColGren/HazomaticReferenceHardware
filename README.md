# Hazomatic Reference Hardware Overview

Hazomatic Reference Hardware is a DIY reference implementation of the
Hazomatic particulate sensor protocol for ESP32-based devices.

This repository includes:
- Precompiled firmware (.bin)
- 3D-printable enclosure files
- Build documentation
- The Hazomatic protocol specification

This project is provided for experimental and educational use.
It is not a certified measurement instrument and is not intended for
regulatory, occupational health, or safety-critical applications.

Source code for the firmware is not included.
